Boss Modules:

These profiles get automatically updated by Twitch Plays.

NoBossModules Disables all modules that are considered Boss Modules.
NoPseudoNeedy Disables all modules that are considered Pseudo Needies.

Needy profiles:

These profiles get automatically updated by Twitch Plays.

No0 Contains all needy modules with a score of 0.
No0+ Contains all needy modules with a score between 0 and 1. (Exclusive, Exclusive).
No1 Contains all needy modules with a score between 1 and 2. (Inclusive, Exclusive).
No2 Contains all needy modules with a score between 2 and 3. (Inclusive, Exclusive).
No3, No4, No5, etc
NoStaticNeedies Contains all needy modules that have a static score that isn't 0.
NoTimeNeedies Contains all needy modules that have a score based on time.
NoOtherNeedies Contains all needy modules that don't fit any of the above listed categories.

Additional profiles:

ON Contains all modules that are broken in some way. Please don't disable this profile unless absolutely necessary. Like for example running a mission. All modules disabled by this profile can be found in the #on-profile channel in the Discord Server.
NoFactoryRoom Disables the Factory gameplay room.
