Twitch Plays Profiles

This site shows a list of all available profiles featured on The Official Twitch Plays KTaNE Stream, along with what module(s)/widgets it disables.

To enable/disable a profile, type "!profiles enable/disable" followed by the profile name in the stream chat.

To request a new profile, send a message in #add-modules in The Official Twitch Plays KTaNE Discord Server

If you find any errors or have any suggestions feel free to contact Marksam in the Discord Server

Your IP address is not logged. No cookies are collected.
